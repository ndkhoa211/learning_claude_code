▶ What Are Output Styles?

Output styles are customizable response templates that control how Claude Code formats and structures its responses to match your workflow preferences.

┌─────────────────────────────────────────────────────────────────────────────┐ │ TECHNICAL DETAILS │ └─────────────────────────────────────────────────────────────────────────────┘

◆ Storage & Structure

▶ Global: ~/.claude/output-styles/

▶ Project: ./.claude/output-styles/

▶ Format: Markdown files with YAML frontmatter

▶ Activation: Via /output-style command
